# Simple-BlockChain-Structure
-----------------------------
A simple implementation of BlockChain.
<br> <br>
采用Java对区块链数据结构的简单实现。
<br><br>
文档中主要包含：
 - MainBlock：区块的基本结构
 - Transaction：交易数据的具体信息，用户可自定义
 - BlockChain：区块链的链式结构
 - BlockChainService：区块链具体操作，包括创建创世区块，创建、添加区块等
 - BlockController：区块链测试类
<br>
将来添加Pow或其他共识算法。
